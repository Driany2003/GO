Resumen

Resume

Desarrollador Full Stack Junior con 2 años de experiencia en el ecosistema Java. Mi expertise abarca el desarrollo backend robusto con Spring Boot, Spring Security y Spring Cloud, y la creación de APIs REST de alto rendimiento. En el frontend, poseo sólidas habilidades en React, JSP, HTML5 y Bootstrap, para construir interfaces de usuario dinámicas y responsivas. Cuento con experiencia práctica en la implementación de arquitecturas escalables en la nube (AWS), dominando servicios clave como S3, EC2, Lambda, Glue y Athena. Mi perfil incluye la capacidad de integrar soluciones IoT y de orquestar pipelines CI/CD para despliegues eficientes y automatizados. Soy un defensor de la seguridad desde el diseño, aplicando principios DevSecOps y utilizando herramientas como Snyk y Jest para garantizar la calidad y protección del software. Además, tengo experiencia en el manejo de bases de datos relacionales (SqlServer, MySQL, PostgreSQL), la realización de pruebas de estrés con Postman, y la aplicación efectiva de metodologías ágiles (Scrum), utilizando herramientas como Jira y Miro para la gestión de proyectos. Mis habilidades se complementan con la capacidad de realizar análisis de datos usando Python y Power BI, transformando información en insights accionables.
